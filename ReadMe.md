### Installation

```bash
git clone https://github.com/furkankayam/file-share.git
```

<br>

### Editing Config

- file-share/frontend/config.js

```bash
// Backend App URL
export const API_URL = "http://<ubuntu_server_ip>:8080";
```

<br>

### Usage

```bash
docker-compose up -d
```

<br>

### URL

```bash
http://<ubuntu_server_ip>:5173
```
